# 🎙️ ESP32-S3 Audio Streaming System
Real-time audio streaming from ESP32-S3 with INMP441 microphone to web browser via WiFi, featuring live waveform visualization and spectrum analysis.

## Componenti
- ESP32
- PC (Windows 11)

## Come usare
1. Carica il file .ino sull'ESP32 con Arduino IDE.
2. Esegui lo script python sul PC: `python script_pc.py`.
3. Apri il file html nel browser.

## Note
Ricordati di inserire le tue credenziali Wi-Fi nel codice prima di compilare.
